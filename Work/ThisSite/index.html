Designing A Professional Website
I've needed a portfolio website for a LONG time. The goal with this one was to make it mobile friendly, cross-platform, and intuitive. All of the code is custom and I didn't use a template or anything like that. All of the designs are my own. If you want to know more, just look around! There is a lot to see. The home page display boxes took a good bit of time. They are dynamically placed when the page loads. This will allow me to add more in the future. I had to create an algorithm to place them all without any blank space in a way that looks good. Again, I feel the website speaks for itself about the extent of my web design skills. Show, not tell, as they always say!
