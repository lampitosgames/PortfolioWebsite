About the Site
My name is Daniel, I work with a lot of code and UI design, and this is a website I created for my Intro to Interactive Media class. The site is based around an action adventure RPG called Dark Souls, a fantastic game built with fantastic design and forethought. If you look around, you can read about why I love this game so much.
Web design is something I love doing and hope to continue throughout my entire career. The only javascript API I used for this site is JQuery, the rest is custom code. Custom parallaxing code is not something I see very often, so I decided it might be fun to try! I was right, I had a blast, and it works seamlessly across browsers.
