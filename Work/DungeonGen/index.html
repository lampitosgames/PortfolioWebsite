Semi-Random Dungeon Generation
One of my current game development projects involves generating dungeon levels from a set of pre-built rooms. I want to get users creating their own levels, so I needed a way to easily import new parts of the dungeon. Rooms can be created externally and dropped in the game folder. Anything from large puzzles to simple hallways. The generator takes all of that information and strings rooms together to create a floor of the dungeon. Rooms can be given spawn rarity, any size, and any number of doorways. All of this was done in GameMaker, and seeing as it is an active project that I want to sell eventually, I'm not going to be providing the source code.
